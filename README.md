# 🛡️ SecureTool - Advanced Code Analysis & Encryption

SecureTool, Python ile geliştirilmiş, statik kod analizi (SAST) ve yüksek güvenlikli dosya şifreleme özelliklerini bir arada sunan bir güvenlik aracıdır.

## 🚀 Özellikler
* **Kod Analizi:** SQL Injection, XSS, API anahtarı sızıntıları ve zayıf şifreleme tespiti yapar.
* **Güvenli Şifreleme:** Argon2id ve 3 katmanlı AES-256-GCM kullanarak dosyalarınızı korur.
* **Raporlama:** Analiz sonuçlarını detaylı bir JSON dosyası olarak kaydeder.

## 🛠️ Kurulum & Kullanım
1. `pip install cryptography` komutuyla gerekli kütüphaneyi yükleyin.
2. `python securetool.py` komutuyla aracı başlatın.
